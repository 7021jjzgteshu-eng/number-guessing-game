# 数当てゲーム (Number-Guessing-Game)

Pythonで作った簡素な数当てゲームです。

## 現在の機能と遊び方
1. 難易度を選択できるので、1~4(5)で選んでください。
2. 制限回数があるので、ヒントを元に制限以内にコンピュータが選んだ数字を当ててください。
3. 終了時のコメントが出た後、リトライするか選択できます。
